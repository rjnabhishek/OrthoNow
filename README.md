# OrthoNow GTM & GA4 Tracking Assignment

## Overview

This project demonstrates the implementation of Google Tag Manager (GTM), Google Analytics 4 (GA4), event tracking, and a responsive landing page for OrthoNow, a multi-clinic orthopedic healthcare provider.

## Features

- Responsive landing page
- Multi-step appointment booking flow
- GTM implementation
- GA4 event tracking
- Custom dataLayer events
- Google Ads conversion tracking recommendations
- Netlify deployment

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Google Tag Manager
- Google Analytics 4
- Netlify
